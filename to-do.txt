[ ] PROD-0001 — IT Operations Automation SaaS Platform Setup
------------------------------------------------------------------------------
  Priority: HIGH
  Dependencies: None
  Release: None

  DESCRIPTION:
  Set up the foundational architecture for a multi-tenant SaaS platform
  that automates IT operations. This task covers project scaffolding,
  core infrastructure, authentication, database schema, and the initial
  workflow engine. The platform targets SMBs and mid-market companies
  needing to automate repetitive IT tasks like user onboarding/offboarding,
  software provisioning, scheduled maintenance, ticket routing, and
  compliance checks. Built as a subscription-based service with a visual
  workflow builder as the primary user interface.

  TECHNICAL DETAILS:
  Frontend:
    - Next.js 14+ with TypeScript and App Router
    - Tailwind CSS for styling
    - React Flow for visual workflow builder canvas
    - Zustand for client-side state management

  Backend:
    - Node.js with Fastify and TypeScript
    - PostgreSQL with Prisma ORM
    - Redis for caching and job queues
    - BullMQ for background workflow execution
    - JWT + refresh tokens for authentication

  Infrastructure:
    - Docker + docker-compose for local development
    - Multi-tenant database design (row-level security)
    - Environment configuration (.env templates)

  Core Modules:
    - Auth service (registration, login, org management)
    - Workflow engine (create, schedule, execute, monitor)
    - Template library (pre-built IT automation templates)
    - Execution runtime (step runner, error handling, retries)
    - Dashboard (workflow status, execution history, analytics)
